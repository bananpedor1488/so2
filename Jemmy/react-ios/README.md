# React Native iOS проект

## Установка

1. Установите зависимости:
```bash
cd react-ios
npm install
```

2. Установите iOS зависимости:
```bash
cd ios
pod install
cd ..
```

3. Запустите Metro bundler:
```bash
npm start
```

4. Откройте `ios/ReactIOS.xcworkspace` в Xcode и запустите проект

## Структура
- `App.js` - главный компонент приложения
- `index.js` - точка входа
- `ios/` - нативный iOS проект для Xcode
